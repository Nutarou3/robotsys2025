～OOO.kadaiの使い方～

おなかがすいたけど献立が決まらない！！そんなことありますよね。本プログラムは、そんなあなたの悩みを解決します。

1-1.プログラム内容およびジャンルの説明

本コマンド OOO,kadai は、標準入力で与えられたジャンル名に重み付けを行い、その重みに基づいて献立を確率的に選定し、出力するデータフィルタコマンドです。
ジャンルには以下の選択肢があります。
- 食材系: にく, さかな
- 調理法系: 揚げ物, 煮物, 炒め物
- 米飯・麺類系: 米, パン, パスタ
- 国・地域: 和食, 中華, イタリアン, ベジタリアン
- その他: ジャンクフード

1-2.動作環境

OS: WSL (Ubuntu) または Linux 環境

言語: Python 3.8 以上

1-3.使用方法

OOO.kadai をコマンドとして実行できるようにパーミッションを付与します。
$ chmod +x OOO.kadai
実行の際には以下のように実行してください
$ ./OOO.kadai にく(ジャンル名)
