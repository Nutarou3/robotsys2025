## menu_proposer

本リポジトリは、千葉工業大学のロボットシステム学の課題用リポジトリです。標準入力から受け取ったジャンル名に基づき、献立を提案するフィルタコマンドを提供します。

## コマンドの概要 

おなかがすいたけれど献立が決まらないとき、このコマンドが解決します。
標準入力からジャンル名のリストを読み込み、その中から一つをランダムに選び、具体的な料理名を出力します。

### プログラム内容およびジャンルの説明
**特徴：重み付けサンプリング**

入力されるデータの重複を「重み（優先度）」として扱います。
- 例: 「にく」を2回、「さかな」を1回入力した場合、統計的に3回中2回の割合で肉料理が選ばれるようになります。

**ジャンルには以下の選択肢があります。**
- 食材系: にく, さかな
- 調理法系: 揚げ物, 煮物, 炒め物
- 米飯・麺類系: 米, パン, パスタ
- 国・地域: 和食, 中華, イタリアン, ベジタリアン
- その他: ジャンクフード

## 動作環境

OS: WSL (Ubuntu) または Linux 環境

言語: Python 3.8 以上

## 使用方法

**1. 準備（実行権限の付与）**

ファイルに実行権限がない場合は,menu_proposer をコマンドとして実行できるようにパーミッションを付与します。
```
$ chmod +x menu_proposer
```
**2.実行例**
実行の際には以下のように実行してください
```
$ echo ジャンクフード | ./menu_proposer
```
実行結果は以下のようになります。
```
にく:ハンバーグ
```
**2.実行例2**
入力例（重複させて確率を変える場合）
```
$ echo -e "にく\n中華\nにく" | ./menu_proposer
```
実行結果は以下のようになります。
```
にく:焼肉
```

※　実際に使う際にはジャンクフードの位置に、適切なジャンルを入力してください。
